How to play



x, y: Represent the position,
values allowed are 0, 1, 2, 3, 4.

f: Direction that the robot is facing,
values allowed are WEST, EAST, NORTH, SOUTH.

Enter all values and press Place

Right turns the robot to the right, Left turns it left.

Move moves the robot in the direction he is facing.

Report reports the position and facing.

Last but not least,
You can use the arrow keys on you'r keyboard as well! :)
